# Spatial Video Converter

Конвертер пространственных видео (Spatial Video) от iPhone в формат Side-by-Side (SBS) для просмотра в VR-плеерах.

## 🎯 Возможности

- ✅ Конвертация MV-HEVC (Spatial Video) в SBS формат 3840×1080
- ✅ Два пути декодирования: AVFoundation (быстро) → ffmpeg (fallback)
- ✅ GPU-ускоренный композитинг через Metal/CoreImage
- ✅ Поддержка H.264 и H.265 кодеков
- ✅ Настройки качества (высокое / среднее / компактное)
- ✅ Сохранение оригинального аудио (AAC passthrough)
- ✅ Пакетная обработка файлов
- ✅ Drag & drop интерфейс
- ✅ Автоматическое именование `{имя}_SBS_LR.mp4` для DeoVR

## 📋 Требования

- **macOS 14.0+** (Sonoma или новее)
- **Xcode 15+** для сборки
- **ffmpeg** (опционально, для fallback пути)

### Установка ffmpeg

```bash
brew install ffmpeg
```

## 🚀 Сборка

1. Откройте проект в Xcode
2. Выберите цель macOS (My Mac)
3. Нажмите ⌘R для запуска

## 💡 Использование

### Способ 1: Drag & Drop
- Перетащите `.mov` или `.mp4` файлы в окно приложения
- Или перетащите на иконку приложения в доке

### Способ 2: Выбор файлов
- Кликните по зоне drop или "Добавить ещё"
- Выберите Spatial Video файлы

### Настройки

**Выходная папка:**
- Рядом с исходным файлом (по умолчанию)
- Или выберите другую папку

**Видеокодек:**
- H.264 — максимальная совместимость
- H.265 — меньший размер файла

**Качество:**
- Высокое: ≈35 Мбит/с (H.264) / ≈20 Мбит/с (H.265)
- Среднее: ≈20 Мбит/с / ≈12 Мбит/с
- Компактное: ≈10 Мбит/с / ≈6 Мбит/с

## 📂 Формат вывода

- **Разрешение:** 3840×1080 (Full SBS)
- **Формат:** MP4
- **Именование:** `{оригинал}_SBS_LR.mp4`
- **Совместимость:** DeoVR, Skybox VR, другие SBS плееры

## 🛠️ Архитектура

```
┌─────────────────┐
│ ContentView     │ ← UI (SwiftUI)
└────────┬────────┘
         │
┌────────▼────────┐

└────────┬────────┘
         │
┌────────▼─────────────┐

└──┬────────────────┬──┘

```

## 🔍 Работа с файлами

### После конвертации:
1. **Правый клик** на файл → "Показать в Finder"
2. Файл автоматически называется `{имя}_SBS_LR.mp4`
3. DeoVR автоматически распознает формат по суффиксу `_LR`

### Индикаторы:
- ✅ Зелёная галочка — успешно
- 🟧 Метка "ffmpeg" — использован резервный путь
- ❌ Красный крестик — ошибка (наведите для деталей)

## 🐛 Решение проблем

### AVFoundation не может извлечь стереовиды
- Приложение автоматически переключится на ffmpeg
- Появится оранжевая метка "ffmpeg" при завершении

### ffmpeg не найден
```bash
brew install ffmpeg
```

### Ошибка "Не пространственное видео"
- Убедитесь, что видео снято на iPhone 15 Pro или новее
- В режиме Spatial Video (Настройки → Камера)
